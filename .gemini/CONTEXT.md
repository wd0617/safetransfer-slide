# SafeTransfer - Ecosystem Overview

## Related Projects

This project is part of the **SafeTransfer ecosystem**, which consists of multiple interconnected applications:

### 1. SafeTransfer Clientes
- **Location**: `C:\Users\Wander\Desktop\SafeTransfer clientes`
- **Purpose**: Main client-facing web application for money transfer businesses
- **Tech Stack**: React + TypeScript + Vite + Supabase + TailwindCSS
- **Features**: Client management, transfer tracking, compliance (KYC/AML), multi-language (IT/ES/EN), Chrome extension for data capture
- **URL**: https://safetransfer-pi.vercel.app / https://safetransfer.it

### 2. SafeTransfer Slides (This Project)
- **Location**: `C:\Users\Wander\Desktop\SafeTransfer Slides`
- **Purpose**: Presentation/slides application for SafeTransfer (sales, onboarding, documentation)
- **Tech Stack**: React + TypeScript + Vite + Supabase + TailwindCSS
- **Relationship**: Shares the same Supabase backend and branding with SafeTransfer Clientes

### 3. SafeTransfer Chrome Extension
- **Location**: `C:\Users\Wander\Desktop\SafeTransfer clientes\SafeTransfer-Extension`
- **Purpose**: Chrome extension that captures client/transfer data from Western Union, Ria, MoneyGram, Mondial Bony, Monty
- **Relationship**: Sends captured data to SafeTransfer Clientes app

## Shared Resources
- **Supabase project**: Both apps share the same Supabase backend
- **Branding**: Same brand identity, colors (blue/green theme), logo
- **Language**: Primary language is Italian, supports Spanish and English
- **Target users**: Money transfer businesses (agenzie di money transfer) in Italy
